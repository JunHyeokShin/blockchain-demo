사용된 기술

1. HTML

- 시맨틱 태그(header, nav, section)로 구조화.

2. CSS

- Grid로 반응형 레이아웃 설계.
- 미디어 쿼리로 모바일/데스크톱 최적화.
- 호버 애니메이션으로 사용자 경험 개선.

3. JavaScript

- 모바일 메뉴 토글, 데모 페이지의 블록체인 시뮬레이션.
- CryptoJS 라이브러리를 활용하여 SHA-256 해시 계산.
